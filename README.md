## Deskripsi Proyek
**BookTime Store** adalah aplikasi mobile berbasis Flutter yang memungkinkan pengguna untuk mencari, menjelajahi, dan melihat detail buku dari seluruh dunia secara real-time. Aplikasi ini memanfaatkan Google Books API untuk menyajikan data buku yang akurat dan up-to-date.

## Fitur Utama
- **Integrasi Google Books API**: Pencarian buku secara langsung dari database Google.
- **Kategori Populer**: Filter buku berdasarkan kategori seperti Fiction, Philosophy, Business, dll.
- **Pencarian Real-time**: Cari buku favorit Anda melalui kolom pencarian.
- **Detail Buku**: Informasi lengkap mengenai penulis, deskripsi, harga, dan rating.
- **Keranjang Belanja (Simulasi)**: Tambahkan buku yang diinginkan ke keranjang belanja.
- **Desain Responsif**: Antarmuka yang bersih dan modern menggunakan Material 3, mendukung berbagai ukuran layar.

## Teknologi yang Digunakan
- **Flutter SDK**: Framework utama.
- **Dart**: Bahasa pemrograman.
- **HTTP Package**: Untuk komunikasi dengan REST API.
- **Material 3**: Untuk desain UI modern dan dinamis.

## Cara Menjalankan Project
1. Pastikan Anda sudah menginstal Flutter SDK di perangkat Anda.
2. Clone repository ini.
3. Jalankan perintah `flutter pub get` di terminal untuk mengunduh dependencies.
4. Hubungkan emulator atau perangkat fisik.
5. Jalankan aplikasi dengan perintah `flutter run`.
