Project ini dibuat untuk memenuhi tugas mata kuliah Pemrograman Perangkat Bergerak (ABP). Aplikasi ini adalah toko buku digital sederhana yang datanya diambil langsung dari Google Books API.

### Tentang Aplikasi
Aplikasi **BookTime Store** memungkinkan user buat cari buku secara online. Fitur-fiturnya antara lain:
* Cari buku pakai keyword lewat search bar (integrasi Google Books API).
* Filter buku berdasarkan kategori (Fiction, Philosophy, dll).
* Lihat detail buku (deskripsi, harga, rating, penulis).
* Simulasi keranjang belanja (bisa tambah/hapus item).
* Tampilan sudah responsif (bisa menyesuaikan ukuran layar hp/tablet).

### Tech Stack
* Flutter & Dart
* HTTP Package (buat ambil data dari API)
* Material 3 Design

### Cara Menjalankan
1. Pastikan Flutter sudah terinstall.
2. Buka folder project di terminal.
3. Jalankan `flutter pub get`.
4. Connect ke HP atau Emulator.
5. `flutter run`.
